（1）按头排序：从小到大头排序，头同，从小到大尾排序。
头同，则尾大删。
头不同，若新头<原尾，则尾大删。

尾同，则头小删。
尾不同，若新尾<原头，则头小删。
反向思考。求最多的不重叠区间需要移除区间的个数，先求不重叠区间，再减。

排序的时候，规则是尽量往结果序列靠拢。

动态规划：
状态 + 状态转移方程。
某个状态只与它前面出现的状态有关，而独立于后面的状态。

入门题：用数量最少的面值1，3，5的硬币凑够i元：
d(i) = min{ d(i-Vj) + 1 }, 其中i-Vj >= 0。Vj表示第j个硬币的面值。

经典问题：LIS(Longest increasing subsequence) 最长非降子序列：
d(i) = max{ d(j)+1, 1 }, 其中j < i，A[j] < A[i]。
d(i)：把i前面各个子序列中，最后一个数不大于A[i]的序列+1的长度中的最大长度。如果不满足最后一个数<=A[i]，那么只能为1。

记录最小价格min，将其作为买入价格，当前价格为卖出价格计算pro。不断更新min和pro。
动态规划解法：
d(i)的值要么是0，要么是今天的价格-今天之前里面最小价格为买入值。最终返回所有d里面的最大值。
或者边走边比较，0，今天的价格-今天之前里面最小价格，前一天的d，哪个大保存哪个，那么最终最后一个就是最大值。
d(i) = max{ d(i-1), A[i]-min };

d[i] = Math.max(d[i-1], A[i] - min)
d[i] = Math.max(d[i], 0)    //提高runtime
return d[A.length-1]
