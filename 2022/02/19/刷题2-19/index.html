入门题：用数量最少的面值1，3，5的硬币凑够i元：
d(i) = min{ d(i-Vj) + 1 }, 其中i-Vj >= 0。Vj表示第j个硬币的面值。

经典问题：LIS(Longest increasing subsequence) 最长非降子序列：
d(i) = max{ d(j)+1, 1 }, 其中j < i，A[j] < A[i]。
d(i)：把i前面各个子序列中，最后一个数不大于A[i]的序列+1的长度中的最大长度。如果不满足最后一个数<=A[i]，那么只能为1。

d(i) = max{ d(i-1), A[i]-min };

123：
动态规划。
五种状态：
未进行过任何操作； //该状态利润为0，不用记录
只进行过一次买操作； // buy1
进行了一次买操作和一次卖操作，即完成了一笔交易； //sell1
在完成了一笔交易的前提下，进行了第二次买操作； //buy2
完成了全部两笔交易。 //sell2
初始状态：
buy1 = −prices[0]
sell1 = 0
buy2 = −prices[0]
sell2 = 0
状态转移：
buy1 = max{ buy1, −prices[i] }
sell1 = max{ sell1, buy1 + prices[i] }
buy2 = max{ buy2, sell1 − prices[i] }
sell2 = max{ sell2, buy2 + prices[i] }
如果最优的情况对应的是恰好一笔交易，那么它也会因为转移时允许在同一天买入卖出第二笔，从 sell1 转移至 sell2。
return sell2
