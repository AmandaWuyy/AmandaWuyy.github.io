Webpack打包Vue源码还原：

目前找到两种：reverse-sourcemap 和 restore-source-tree

reverse-sourcemap一口气还原，但效果差，缺失文件多

restore-source-tree会一个文件夹一个文件夹的还原

用用restore-source-tree 一次还原source map 网站原始码改过的restore-source-tree

首先寻找和下载sourcemap文件：

f12 → sources → webpack:// → 点开每个文件夹，发现下面有“source mapped from xxx.js”

右键复制这段路径

下载map文件：

curl https://lysh.smgtech.net/js/chunk-vendors.70af97b4.js.map > chunk-vendors.70af97b4.js.map

curl https://lysh.smgtech.net/js/chunk-f260e320.6e283292.js.map > chunk-f260e320.6e283292.js.map

curl https://lysh.smgtech.net/js/app.07fa07b9.js.map > app.07fa07b9.js.map

安装：

npm install -g Yukaii/restore-source-tree#98ccfc2

还原：

restore-source-tree -n --out-dir ./rebuild chunk-vendors.70af97b4.js.map
restore-source-tree -n --out-dir ./rebuild chunk-f260e320.6e283292.js.map
restore-source-tree -n --out-dir ./rebuild app.07fa07b9.js.map
